Helpful Definitions

Functions are a block of reusable code that performs a specific task in Python. They allow you to break your code into smaller, more manageable pieces, making it easier to read, understand, and maintain. Functions in Python are defined using the def keyword, followed by the function name, a list of parameters in parentheses, and a colon. def func_name(params):

In Python, arrays are implemented as lists. A list is a collection of values, separated by commas, enclosed in square brackets []. Lists can contain elements of different data types and can be accessed using index notation. five_fibo = [1,1,2,3,5]

While loops are a control flow statement in Python that allow you to repeatedly execute a block of code as long as a condition is true. while num < 2:
