Helpful Definitions

The for loop is a control flow statement in Python that allows you to iterate over a sequence of values. It is used to execute a block of code repeatedly for each element in the sequence. Eg. for i in sequence:

Dictionaries are a built-in data structure in Python that allow you to store a collection of key-value pairs. Each key is unique and maps to a corresponding value. Eg: hash_map = {'One': 1}

Type casting is the process of converting one data type to another in Python. Python provides several built-in functions for type casting, including int(), float(), str(), list(), tuple(), set(), and dict(). int_one = int('1')
