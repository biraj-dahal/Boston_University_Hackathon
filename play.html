Helpful Definitions

input() is a built-in Python function that allows the user to input a value from the console. It returns a string containing the input entered by the user. user_number = input("Your number?")

print() is a built-in Python function that outputs data to the console. It can be used to display text, numbers, or other types of data. print("Hello World")

If else statement is a conditional statement in Python that allows the program to execute certain code if a condition is true, and a different set of code if the condition is false. i = 1 if True else 0

The random module is a built-in Python module that allows the program to generate random numbers. It provides various functions for generating random numbers, such as random.randint() for generating random integers, and random.random() for generating random floating-point numbers between 0 and 1. Eg: chosen = random.choice([param1,param2])
